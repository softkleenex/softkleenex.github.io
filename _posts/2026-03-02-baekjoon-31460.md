---
풀이 

우선 바로 머릿속에 떠오른것은 n자리 11 배수의 규칙성이다

11  > 11 * 10 + 11 (121) > 121*10 + 10 (1221)....

이런식으로, n 이 3이상이라면 일반화가 가능했고,
n이 2라면 11을 출력
n이 1이라면 -1을 출력하는방식으로 하다가 Fail을 여러차례받았다.
gemini의 도움을 받아 n이 1일때 0을 출력해야하는걸 깨달았다.




---
```python
# https://www.acmicpc.net/problem/31460

import sys
import collections


t = int(input())

12221

11
121

1221


for i in range(t):
    n = int(input())
    if n == 1:
        print(0)
    elif n == 2:
        print(11)
    else:
        print("1", '2' * (n - 2), "1", sep = '')
```
